# cli-player

一个用 Golang 编写的命令行音乐播放器，支持歌词同步显示和可视化终端界面。

## 功能特性

- 🎵 **命令行终端用户界面（TUI）** - 基于 bubbletea 的美观终端界面
- 🎧 **多种音频格式支持** - MP3、FLAC、OGG、WAV
- 📝 **歌词文件显示** - 支持 LRC 格式歌词，自动同步显示
- 📋 **播放列表管理** - 添加、删除、跳转曲目
- 🔀 **播放控制** - 随机播放、单曲循环、列表循环
- 🎚️ **音量调节** - 平滑的音量控制
- ⏯️ **进度控制** - 快进、快退、跳转播放
- 📂 **文件浏览器** - 浏览本地目录选择音乐

## 环境要求

- Go 1.25.0 或更高版本
- 操作系统音频驱动支持（macOS、Linux、Windows）

## 编译方法

### 从源码编译

```bash
# 1. 克隆项目
git clone <repository-url>
cd cli-player

# 2. 下载依赖
go mod download

# 3. 编译
go build -o cli-player .

# 编译完成后，可执行文件为 cli-player
```

### 交叉编译

```bash
# Linux (amd64)
GOOS=linux GOARCH=amd64 go build -o cli-player-linux .

# macOS (arm64)
GOOS=darwin GOARCH=arm64 go build -o cli-player-mac .

# Windows (amd64)
GOOS=windows GOARCH=amd64 go build -o cli-player.exe .
```

## 运行方法

### 基本用法

```bash
# 指定音频文件目录运行（程序会从该目录开始浏览）
./cli-player /path/to/music

# 指定单个音频文件运行（程序会自动定位到文件所在目录）
./cli-player /path/to/song.mp3

# 使用当前目录运行
./cli-player .
```

### 歌词文件支持

将歌词文件与音频文件放在同一目录下，使用相同的文件名：
- 音频文件：`song.mp3`
- 歌词文件：`song.lrc` 或 `song.LRC`

歌词文件格式（LRC 标准）：
```
[00:12.50]第一句歌词
[00:15.80]第二句歌词
[00:18.20]第三句歌词
```

## 界面说明

程序启动后，界面分为左右两个面板：

**左侧面板**
- **播放器区域**：显示当前播放曲目信息、进度条、播放状态、音量
- **播放列表区域**：显示已添加的曲目列表

**右侧面板**
- **文件浏览器**：浏览目录选择音乐文件（默认显示）
- **歌词显示**：显示同步的歌词（当播放有歌词的曲目时自动切换）

### 使用流程示例

1. **启动程序**
   ```bash
   ./cli-player ~/Music
   ```

2. **添加音乐**
   - 使用 `↑`/`↓` 选择目录或音频文件
   - 按 `Enter` 进入目录，或直接按 `Enter` 添加音频文件
   - 也可以按 `a` 仅添加文件而不进入

3. **播放音乐**
   - 使用 `Tab` 切换到播放列表
   - 选择要播放的曲目，按 `Enter` 开始播放

4. **查看歌词**
   - 当播放的曲目有对应的 `.lrc` 文件时，右侧面板会自动显示歌词
   - 歌词会随着播放进度自动滚动
   - 可以使用 `↑`/`↓` 手动浏览歌词，按 `Enter` 跳转
   - 按 `l` 可在歌词和文件浏览器之间切换

5. **控制播放**
   - `Space` 暂停/继续
   - `n`/`p` 切换下一首/上一首
   - `</>` 快进/快退 10 秒

6. **调节音量和模式**
   - `+`/`-` 调节音量
   - `s` 开启/关闭随机播放
   - `r` 切换循环模式

## 常见问题

**Q: 为什么没有声音？**
A: 请检查系统音量设置，确保操作系统的音频驱动正常工作。

**Q: 歌词没有显示？**
A: 确保 LRC 文件与音频文件在同一目录下，且文件名相同（仅扩展名不同）。

**Q: 支持哪些音频格式？**
A: 目前支持 MP3、FLAC、OGG、WAV 格式。

**Q: 如何在 macOS 上获得更好的音频输出？**
A: 可以使用 Homebrew 安装 `portaudio`，beep 库会自动使用系统音频驱动。
